# Repeat

## Examples

```xml
<repeat i="10">
    <rect x="{i*10}" y="0" width="5" height="5" />
</repeat>
```

<br>

## Parameters

### any

**Name** <br>

The name of the counter variable which will be used (Use \_ to specify not using a variable).

**Value** <br>

An expression that returns the number of times the loop will be ran.
