## Installation

Linux:
```bash
git clone https://github.com/KotwOSS/resvg.git
cd resvg
chmod +x resvg.py
sudo ln resvg.py /usr/bin/resvg
```

<hr>
<br>

## Usage

Show the help
```sh
resvg help
```

Show the version
```sh
resvg help
```

Compile a ReSVG file to a SVG file:
```sh
resvg compile input.rsvg output.svg
```
